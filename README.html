Generating docs

This published version of the docs can be refreshed by these commands:

  1. Prepare one off

    dotnet tool restore git clone https://github.com/dsyme/DiffSharp ../DiffSharp-docs -b gh-pages --depth 1

  2. Build

    dotnet build dotnet fsdocs build --clean --output ../DiffSharp-docs bash -c "(cd ../DiffSharp-docs && git add . && git commit -a -m doc-update && git push -f https://github.com/dsyme/DiffSharp gh-pages)"
