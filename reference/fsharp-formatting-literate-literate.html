FSharp.Formatting

Literate Type

Namespace: FSharp.Formatting.Literate

Assembly: FSharp.Formatting


This type provides three simple methods for calling the literate programming tool. The ConvertMarkdown and ConvertScriptFile methods process a single Markdown document and F# script, respectively. The ConvertDirectory method handles an entire directory tree (looking for *.fsx and *.md files).
